# Demo task.

Write python program to solve [quadratic equation](https://en.wikipedia.org/wiki/Quadratic_equation) 
```python
a * x * x + b * x + c == 0 
```
Result should be sorted by ascential order.

__Examples__:
```python

>>> solution(a=1, b=6, c=5)
(-5, -1)
>>> solution(a=1, b=4, c=4)
(-2,)
>>> solution(a=1, b=6, c=45)
()
```
